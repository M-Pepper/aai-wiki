## Dismissible Banners

Dismissible banners are floating announcement boxes that users can close. They persist using localStorage.

### Info Banner

<DismissibleBanner id="example-info" variant="info">
  **New Feature:** Check out our updated asthma management guidelines. <a href="/asthma">Learn more</a>
</DismissibleBanner>

### Warning Banner

<DismissibleBanner id="example-warning" variant="warning">
  **Maintenance Notice:** The site will undergo scheduled maintenance on Saturday, 2 AM - 4 AM EST.
</DismissibleBanner>

### Success Banner

<DismissibleBanner id="example-success" variant="success">
  **Updated:** GINA 2024 guidelines have been incorporated into all asthma content.
</DismissibleBanner>

### Danger Banner

<DismissibleBanner id="example-danger" variant="danger">
  **Critical Update:** Major changes to anaphylaxis treatment protocols. <a href="/anaphylaxis">Review immediately</a>
</DismissibleBanner>

**Usage in MDX:**

```tsx
<DismissibleBanner id="unique-id" variant="info">
  Your message here with <a href="/link">links</a>
</DismissibleBanner>
```

Announcement banners appear at the top of every page and are useful for site-wide alerts.

**Example configuration:**

## Clinical Scenario Example

Here's how these components might be used in a real clinical guideline:

### Acute Asthma Exacerbation Management

<AsideMessage type="danger">
  **EMERGENCY**: If the patient has any signs of life-threatening asthma (altered consciousness, cyanosis, poor respiratory effort, silent chest), immediately call for ICU support and prepare for intubation.
</AsideMessage>

**Initial Assessment:**

<AsideMessage type="note">
  Assess severity using peak flow or FEV1 (% predicted or personal best), respiratory rate, heart rate, oxygen saturation, and ability to speak in full sentences.
</AsideMessage>

**Treatment Protocol:**

1. **Oxygen**: Target SpO2 94-98% (93-97% in children)
2. **Beta-2 agonist**: Salbutamol 5mg via nebulizer or 4-6 puffs via MDI with spacer
3. **Corticosteroids**: Prednisolone 40-50mg PO or IV hydrocortisone 100mg

<AsideMessage type="warning">
  Monitor for signs of deterioration. If no improvement after initial treatment, consider:
  - Ipratropium bromide 500mcg via nebulizer
  - IV magnesium sulfate 2g over 20 minutes
  - Senior clinician review
</AsideMessage>

<AsideMessage type="tip">
  **Best Practice**: Document peak flow before and after treatment to objectively assess response. A &lt;20% improvement suggests severe exacerbation requiring admission.
</AsideMessage>

## Code Blocks with Syntax Highlighting

```python
# Example: Calculate predicted peak flow
def predicted_peak_flow(age, height_cm, sex):
    """
    Calculate predicted peak expiratory flow (PEF) in L/min

    Args:
        age: Age in years
        height_cm: Height in centimeters
        sex: 'male' or 'female'

    Returns:
        Predicted PEF in L/min
    """
    if sex == 'male':
        return (5.48 * height_cm) - (0.041 * age) - 463.1
    else:
        return (3.72 * height_cm) - (0.024 * age) - 316.5

# Example usage
pef = predicted_peak_flow(age=45, height_cm=170, sex='male')
print(f"Predicted PEF: {pef:.0f} L/min")
```
