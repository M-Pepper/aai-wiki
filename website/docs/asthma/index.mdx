# Asthma

Comprehensive clinical reference for asthma diagnosis, classification, and management.

## Overview

Asthma is a chronic inflammatory airway disease characterized by variable respiratory symptoms and airflow limitation. It affects millions worldwide and requires individualized management approaches.
